# OCLM Phase 1 — Historical Development Artifacts

This directory preserves non-canonical development artifacts
created during the formalization and verification of
Oracle–Child Lineage Model (OCLM) Phase 1.

It includes:

- superseded formal-model snapshots;
- pre-fix and experimental models;
- counterexample traces and visualizations;
- pre-update proof-index and manifest backups;
- guardedness probes;
- pre-release integrity records.

## Status

These files are historical research evidence.

They are not the canonical OCLM Phase 1 models, proof results,
or frozen releases.

The current canonical entry point is:

../../README.md

The final Phase 1-C6B model is:

../../model/tamarin/oclm_phase1c6b_generalized_security_composition_contract.spthy

The canonical frozen release is:

../../releases/oclm_phase1c6b_verified_20260717.zip

## Immutability

Historical files are preserved without silent replacement.

Corrections or additions require a new version and updated
integrity records.
